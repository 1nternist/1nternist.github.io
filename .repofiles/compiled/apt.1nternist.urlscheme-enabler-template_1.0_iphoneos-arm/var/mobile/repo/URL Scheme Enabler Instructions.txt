## URL Scheme Enabler Template Instructions

1. Edit control file to reflect your target bundle info
	Package:
	Name:
	Sponsor:
	Description:

2. Edit postinst file where
	$CFBundleName = CFBundleName of your target app. (Info.plist)
		i.e. CFBundleName belonging to IPA Installer is IPA_Installer
	/Applications/AppName.app = The physical name of the app
	CFBundleURLName = com.somecompany.somebundle.scheme (change to reflect your target bundle identifier)
		i.e. com.somecompany.somebundle.scheme --> com.surgail.ipainstaller.scheme
	"CFBundleURLSchemes":["bundle-urlscheme"] = Choose your URL Scheme
		i.e. "CFBundleURLSchemes":["bundle-urlscheme"] --> "CFBundleURLSchemes":["ipa-installer"]

3. Edit prerm file where
	$CDBundleName = CFBundleName of your target app. (Info.plist)
		i.e. CFBundleName belonging to IPA Installer is IPA_Installer

4. Edit the MobileSubstrate dylib & plist files within the template to reflect your target app.
	i.e. "${TemplateDirectory}"/Library/MobileSubstrate/DynamicLibraries/BUNDLE_NAME_Enhancer.dylib --> "${TemplateDirectory}"/Library/MobileSubstrate/DynamicLibraries/IPA_InstallerEnhancer.dylib
	i.e. "${TemplateDirectory}"/Library/MobileSubstrate/DynamicLibraries/BUNDLE_NAME_Enhancer.plist --> i.e. "${TemplateDirectory}"/Library/MobileSubstrate/DynamicLibraries/IPA_InstallerEnhancer.plist
